# Conduit Terminal

Conduit Terminal is a Windows desktop SSH client for saved hosts, host groups, split-pane terminals, SFTP, WSL shells, jump hosts, themes, broadcast input, and encrypted credential storage.

Website: https://conduitterminal.github.io/

Download: https://conduitterminal.github.io/downloads/Conduit%20Setup%201.0.0.exe

## Download

The current public Windows installer is:

```text
downloads/Conduit Setup 1.0.0.exe
```

Conduit Terminal is free for personal use. Enterprise usage is $10/month, billed yearly at $120/year.

## Features

- Saved SSH hosts and host groups
- Split-pane terminal workspace
- Broadcast input across panes
- SFTP browse, upload, and download
- Local WSL shell sessions
- Jump host support
- App and terminal themes
- Encrypted credential storage when supported by the operating system

## Release Updates

The app checks the release policy and shows a manual download banner when a newer Windows installer is available. It does not silently auto-update or reinstall.
